üíì IoT-Based Health Monitoring System

Overview

A comprehensive wearable health monitoring system that continuously tracks vital signs including heart rate, SpO2, body temperature, and sends data to cloud for remote monitoring by healthcare professionals.

Key Features

  • MAX30102 pulse oximeter for heart rate and SpO2 measurement
  • MLX90614 non-contact temperature sensor
  • ESP32 for WiFi connectivity and data transmission
  • Real-time alerts via GSM module for critical conditions
  • Cloud dashboard for doctors and caregivers
  • Low-power design with battery backup

Technologies Used

ESP32, MAX30102, MLX90614, GSM Module, ThingSpeak/Firebase, Arduino IDE, C/C++
